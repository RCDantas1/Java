# Lista de Exercícios 
1- Crie uma classe Animal que permita armazenar as 
informações: 
Nome, raça, peso, altura e idade 

Crie 3 objetos da classe Animal criada anteriormente e 
defina os valores para todos os atributos da classe. 

2- Crie uma classe chamada Operações (sem os acentos), que 
contenha: 
Os atributos: numero1 e numero2. 
Os métodos: soma, subtração, multiplicação e divisão (sem os 
acentos) 

No método main, efetue a leitura dos números 1 e 2.  
Crie um objeto da classe Operações e chame os métodos 
criados passando os números lidos como parâmetro.
